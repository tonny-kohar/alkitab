Study Bible

This table describes the toolbar icons [from left to right].

Button Name Description
Back Previous Page & Next Page, historically.
Forward
Previous Only active if the displayed passage spans more than the range of the scroll bar eg: result of searching
Next
Context Expand 1 Adds 1 verse to each end of currently selected verses
Context Expand 5 Adds 5 verses either end of currently selected verses
Refresh Refresh the current view.
View Options Opens the view options on the currently selected window.
note: to set the viewing options of all new windows use Menu-Tools-Options

*Checkbox "compare", is only applies when using a parallel view in the same language eg: English:- 'KJV' Vs 'ESV'
